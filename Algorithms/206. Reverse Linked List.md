# 206. Reverse Linked List
```
ListNode* reverseList(ListNode* head)
{
    ListNode *prev, *current, *next;

    if (head == NULL) return NULL;

    prev = NULL, current = head, next = head->next;

    while (next)
    {
        current->next = prev;
        prev = current;
        current = next;
        next = current->next;
    }
    current->next = prev;

    return current;
}
```
