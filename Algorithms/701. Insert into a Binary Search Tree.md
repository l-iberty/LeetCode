# 701. Insert into a Binary Search Tree
```
TreeNode* insertIntoBST(TreeNode* root, int x)
{
    if (root == NULL)
    {
        root = new TreeNode(x);
        return root;
    }

    if (x < root->val)
    {
        root->left = insertIntoBST(root->left, x);
    }
    else if (x > root->val)
    {
        root->right = insertIntoBST(root->right, x);
    }
    return root;
}
```
