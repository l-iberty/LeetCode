# 153. Find Minimum in Rotated Sorted Array
```
// Approach 1: Binary Search
int findMin(vector<int>& nums)
{
    int left = 0, right = nums.size() - 1;

    if (nums[right] > nums[0]) // already sorted
        return nums[0];

    while (right - left > 1)
    {
        int mid = (left + right) / 2;
        if (nums[mid] < nums[0])
        {
            right = mid;
        }
        else
        {
            left = mid;
        }
    }

    return nums[right];
}
```
